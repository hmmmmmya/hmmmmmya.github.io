自用vscode插件推荐

  • 中文

    • Chinese (Simplified) (简体中文) Language Pack for Visual Studio Code

  • 主题

    • Material Icon Theme (图标)
    • Night Owl (主题)
    • One Dark Pro (主题)

  • C++

    • C/C++ Extension Pack
      • 包含了C++最受欢迎的扩展, 如C/C++, C/C++ Themes, CMake, CMake Tools
    • C/C++
      • C++ 编译、调试 (需提前安装编译工具, 如MinGW)
    • CMake
      • 语法高亮, 代码补全
    • CMake Tools
      • CMake工具包

  • 文件路径

    • Path Intellisense

  • Git

    • GitLens (查看修改, Git blame)

  • SVN

    • SVN
    • SVN Lens (SVN blame)

  • 项目管理

    • Project Manager (项目之间快速切换, 支持标签)

  • 远程

    • Remote - SSH

  • 代码补全

    • Codeium (支持多行代码补全)

  • Markdown

    • Markdown All in One (markdown快速编辑)
    • markdownlint (markdown语法检查、格式化)

  • 格式化

    • Prettier - Code formatter
    • markdownlint (markdown)

  • Docker

    • Docker
